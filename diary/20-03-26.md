### Solve
In this challenge, I was given a hash and needed to recover the original password.
At first i was given a hash, and i knew hashes are one way funtions but then i was confused that if they are one-way, then how can i get the original password from its hash.

then i read the hint and it told me about this file called Rockyou.txt 
rockyou.txt is a famous password wordlist containing millions of common passwords. Apparently there was a data leak with major passwrods that are commonly used were leaked so we can hash them and check for the hash given to us.

Then learned about *John the Ripper*, an open-source password cracking tool.
but for this I need to know what kind of hash has been put on these,
i know SHA256 returns 64 characters (256 bits)
Basic workflow:
```bash
nano hash.txt
john --wordlist=/path/to/rockyou.txt hash.txt
```
An easier way to do this is through a website called CrackStation which identifies which hash it is as well, and compares it to a database of passwords. It almost feels like cheating. I wonder if using tools like these is allowed in competetions
[CrackStation](https://crackstation.net/)
