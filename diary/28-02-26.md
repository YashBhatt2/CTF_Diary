### Solve
I was given a huge logs file, i had no idea what to do, i foolishly opend the file and it lagged out the 	opened window, after this, i looked at the hint, it said that the text is a base64 string which can be 	converted into an image, so i decoded it:
```bash
cat logs.txt | base64 -d > decoded.txt
```
">" makes the output go to a file
then i renamed the decoded.txt to decoded.png and it was this cool image, that was so cool
there was a hexstring at the bottom, so then i converted it into normal string:
```bash
	echo “the hex string” | xxd -r -p
```
