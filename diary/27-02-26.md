### Solve
it was my first time dealing with an image so i had no idea what to do, apparently i should always approach 	it like this.. 
```bash
#first check the file:
file img.jpg
#then after confirming it is what you think it is, 
#look at the meta data:
exiftool img.jpg
```
then i found a comment of a base 64 thing so i decoded that and inside that there was: 		STEGHIDE(and then another base 64 thing) so i decoded that, and found out that the thing says: 		pAzzword
	
But what is **Steghide**? 
    [Document](https://medium.com/@ece11106.sbit/steghide-tool-ec74edd69de4)
	there is this thing called steganography:Steganography is the practice of concealing a secret 		message, file, or data within an ordinary, non-secret file (like an image, audio, or video) to avoid detection

    and steghide is a tool that enables us to do so, so the flag was embedded in the image and we used 		steghide to extract it
```bash
steghide extract -sf img.jpg
```
I dont know why exactly it writes the data to flag.txt even though nowhere in the command I mention 	that it should do that.
