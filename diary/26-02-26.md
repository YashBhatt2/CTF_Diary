### Learned about **rigrep (rg)**
[Documentation](https://github.com/BurntSushi/ripgrep/blob/master/GUIDE.md)

rigrep is a better and faster version of grep, used to find text within files the best use case for ctf is to just search for flag or things related to the flag:
```bash
rg "picoCTF{"
```
or we can extracts all the strings from a file and pipe them into the command rg:
```bash
strings file.txt|rg -n "picoCTF{"
```
-n show the line number where the string was found, so this is important to have in the command

### Learned about **Base-64 Strings**
How to identify Base64 strings?
    if it ends with = or == (doesnt have to end with one)
    char set is A-Z, a-z, 0-9, +, /, =

How to decode one without the website?
```bash
echo "cGljb0NURg==" | base64 -d
```
"cGljb0NURg==" means pico ctf! so if some string starts like this, then its the flag, but theres no point remembering this since it needs to be decoded anyways.
